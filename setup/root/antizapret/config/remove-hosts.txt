# Исключение доменов из обработки в AntiZapret VPN
# Исключить можно только скачанные или добавленные в config/include-hosts.txt и config/exclude-hosts.txt
#
# Формат записи: example.com
# Где:
#   example.com - доменное имя в кодировке ASCII или Punycode
#
# Примеры записи:
#   subdomain.example.com  - исключение только домена третьего уровня subdomain.example.com
#   example.com            - исключение только домена второго уровня example.com
#   com                    - исключение только домена верхнего уровня com
#   xn--80aswg.xn--p1ai    - исключение только домена второго уровня сайт.рф (кодировка Punycode)
#
# Строки начинающиеся с # это комментарии и они не обрабатываются
#
