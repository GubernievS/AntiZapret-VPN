# Брендирование CorpWeb

Веб-интерфейс CorpWeb поддерживает кастомизацию логотипа для соответствия корпоративному стилю вашей компании.

## Что можно изменить

### 1. Логотип на странице входа

Основной логотип отображается на странице авторизации и в header после входа.

**Расположение файла:** `frontend/public/branding/logo.svg`

**Рекомендации:**
- Формат: SVG (предпочтительно), PNG
- Размер: 200x60 пикселей
- Цвета: Любые, но рекомендуется использовать корпоративные цвета
- Прозрачный фон: Да (для PNG)

### 2. Favicon (иконка сайта)

Маленькая иконка, отображаемая во вкладке браузера.

**Расположение файла:** `frontend/public/branding/favicon.svg`

**Рекомендации:**
- Формат: SVG (предпочтительно), ICO, PNG
- Размер: 32x32 пикселей
- Простой дизайн: Должен быть хорошо различим при малом размере

### 3. Название компании (опционально)

Можно настроить отображаемое название компании в .env файле:

```bash
COMPANY_NAME=Ваша Компания
```

## Как заменить логотип

### Вариант 1: Перед установкой (рекомендуется)

1. Замените файлы в `frontend/public/branding/`:
   ```bash
   cp ~/your-logo.svg frontend/public/branding/logo.svg
   cp ~/your-favicon.svg frontend/public/branding/favicon.svg
   ```

2. Запустите установку:
   ```bash
   ./install.sh
   ```

### Вариант 2: После установки (Docker)

```bash
# Замените логотип в running контейнере
docker cp your-logo.svg corpweb-nginx:/usr/share/nginx/html/branding/logo.svg
docker cp your-favicon.svg corpweb-nginx:/usr/share/nginx/html/branding/favicon.svg

# Перезагрузите nginx
docker exec corpweb-nginx nginx -s reload
```

### Вариант 3: После установки (Прямая установка)

```bash
# Замените файлы
sudo cp your-logo.svg /opt/corpweb/frontend/branding/logo.svg
sudo cp your-favicon.svg /opt/corpweb/frontend/branding/favicon.svg

# Перезагрузите nginx
sudo systemctl reload nginx
```

## Примеры логотипов

### SVG (рекомендуется)

SVG файлы масштабируются без потери качества и выглядят четко на любых дисплеях, включая Retina.

**Пример минималистичного логотипа:**

```svg
<svg width="200" height="60" viewBox="0 0 200 60" xmlns="http://www.w3.org/2000/svg">
  <text x="10" y="40" font-family="Arial, sans-serif" font-size="32" font-weight="bold" fill="#1E40AF">
    YourCompany
  </text>
</svg>
```

### PNG

Для использования PNG убедитесь, что:
- Фон прозрачный
- Разрешение достаточное (минимум 200x60, лучше 400x120 для Retina)
- Сжатие оптимизировано (используйте TinyPNG или подобные инструменты)

## Цветовая схема

Текущая цветовая схема использует синие оттенки:
- Primary: `#3B82F6` (Blue-600)
- Dark: `#1E40AF` (Blue-800)

Вы можете адаптировать свой логотип под эту схему или изменить цвета в `tailwind.config.js`:

```js
theme: {
  extend: {
    colors: {
      primary: {
        600: '#YOUR_COLOR',  // Замените на ваш основной цвет
        800: '#YOUR_DARK_COLOR',  // Замените на темный оттенок
      },
    },
  },
}
```

## Проверка результата

После замены логотипа:

1. Откройте браузер в режиме инкогнито (чтобы обойти кэш)
2. Перейдите на https://vpn-admin.yourcompany.com
3. Проверьте:
   - ✓ Логотип отображается на странице входа
   - ✓ Favicon отображается во вкладке браузера
   - ✓ Логотип четкий и не размытый
   - ✓ Цвета соответствуют корпоративному стилю

## Fallback логотип

Если ваш логотип не загружается, система автоматически отобразит иконку щита (ShieldCheck) из Lucide Icons в качестве резервного варианта.

## Поддержка

Если у вас возникли проблемы с кастомизацией логотипа:

1. Проверьте путь к файлу
2. Убедитесь, что формат поддерживается (SVG, PNG)
3. Проверьте права доступа к файлам (должны быть читаемыми)
4. Очистите кэш браузера

## Примеры успешной кастомизации

- Логотип компании с корпоративными цветами
- Монохромный логотип для минималистичного стиля
- Комбинированный логотип (иконка + текст)
- Только текстовый логотип с корпоративным шрифтом

---

**Важно:** Логотип является частью вашего корпоративного стиля. Убедитесь, что у вас есть права на использование логотипа и он соответствует брендбуку вашей компании.
